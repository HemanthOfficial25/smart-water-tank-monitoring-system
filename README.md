# Smart Water Tank Monitoring System using ESP32

## Overview
This project monitors water levels inside a tank using an ultrasonic sensor and ESP32.

## Features
- Water Level Monitoring
- Tank Overflow Detection
- Low Water Alerts
- Real-Time Data

## Components
- ESP32
- HC-SR04 Ultrasonic Sensor
- Breadboard
- Jumper Wires

## Working
The ultrasonic sensor measures the distance between the sensor and the water surface. ESP32 processes the data and determines the water level inside the tank.
